### 常用快捷键

| 快捷键            | 描述                                                      |
| ----------------- | --------------------------------------------------------- |
| J                 | 删除换行符                                                |
| U                 | 撤销行上所有操作，再次按下则是反撤销                      |
| O                 | 当前行上方插入                                            |
| ZZ                | 保存并退出                                                |
| e!                | 恢复源文件并继续编辑                                      |
| Ctrl-]            | (帮助文档)进入子主题                                      |
| Ctrl-T            | (帮助文档)返回上一级                                      |
| Ctrl-O            | (帮助文档)回到初始的地方                                  |
| d + i + ['"{(...] | 删除对应区间内的内容(d in ...)                            |
| jk                | 从编辑器模式到Normal模式(jdhao/better-escape.vim)         |
| g + Ctrl + g      | 显示文本统计情况                                          |
| Ctrl + g          | 显示当前文件全路径                                        |
| cs                | surround快捷键，快速替换包围符                            |
| cst               | surround快捷键，替换标签                                  |
| ds                | surround快捷键，移除包围符                                |
| ys                | surround快捷键，插入包围付可以使用(ys3j", ysiw', ys2w{)等 |
| yssb, yss)        | surround快捷键，整行插入圆括号                            |
| zz                | 当前行移动到屏幕中央                                      |
