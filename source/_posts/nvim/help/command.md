### 常用快捷键

| 快捷键     | 描述             |
| ---------- | ---------------- |
| \<Left\>   | 向左移动         |
| \<Right\>  | 向右移动         |
| \<Right\>  | 向右移动         |
| \<S-Left\> | 向左移动一个单词 |
| Ctrl-B     | 起始处           |
| Ctrl-E     | 结尾处           |
| Ctrl-W     | 删除一个单词     |

### 替换模式 ❌ (实际尝试是出现一个奇怪的字符 然后没法替换...)

键入<kbd>Insert</kbd>进入替换模式，再次键入则退出替换模式
