## 快捷键

| 快捷键             | 描述                       |
| ------------------ | -------------------------- |
| <kbd>Alt-N</kbd>   | 显示通知中心               |
| <kbd>Alt-A</kbd>   | 显示启动台                 |
| <kbd>F1</kbd>      | 降低显示器亮度             |
| <kbd>F1</kbd>      | 提高显示器亮度             |
| <kbd>Alt-D</kbd>   | 显示桌面                   |
| <kbd>Ctrl-8</kbd>  | 焦点移动到菜单栏           |
| <kbd>Win-`</kbd>   | 将焦点移动到下一窗口(屏幕) |
| <kbd>Ctrl-F3</kbd> | 将焦点移动到程序坞         |
