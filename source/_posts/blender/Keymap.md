### 全局快捷键

|       快捷键        |                              描述                              |
| :-----------------: | :------------------------------------------------------------: |
|       Ctrl-O        |                            打开文件                            |
|       Ctrl-S        |                            保存文件                            |
|    Shift-Ctrl-S     |                             另存为                             |
|       Ctrl-N        |                              新建                              |
|       Ctrl-Z        |                              撤销                              |
|    Shift-Ctrl-Z     |                              重做                              |
|       Ctrl-Q        |                              退出                              |
|         F1          |                              帮助                              |
|         F2          |                             重命名                             |
|         F3          |                            操作搜索                            |
|       F4(🤕)        | 文件上下文(应该是指文件菜单的上下文吧？？？咋想不起还有这操作) |

|  Ctrl-Alt-Spacebar  |                            窗口全屏                            |

### 通用编辑器快捷键

| 快捷键 |    描述    |
| :----: | :--------: |
|   A    |    全选    |
| Alt-A  |  清空已选  |
| Ctrl-I |    反选    |
|   H    |  隐藏所选  |
| Alt-H  | 显示隐藏项 |
|   T    | 切换工具栏 |
|   N    | 切换侧边栏 |
